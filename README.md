## Project Overview
This repository contains the implementation for Homework 2 (HW2). The project demonstrates core front-end development concepts, including modular CSS, layout design, specificity, and basic JavaScript interactivity.

## Key Features & Components
* **Global Styling (`styles.css`):** Implements a CSS reset, custom variables (tokens), fluid typography, and global layout rules.
* **Specificity Exercise (`specificity.html` / `specificity.txt`):** Answers and demonstrations for the Lab 4 specificity exercise, highlighting how the cascade and selector weights apply.
* **Business Card UI (`card.html` / `card.css`):** Demonstrates the use of element, class, ID, and `nth-child` selectors.
* **Interactive Chatbot (`chatbot.html`, `chatbot.css`, `chatbot.js`):** A front-end Q&A interface with simulated delay, keyword-matching logic, and a responsive grid layout.
* **Figma to Code (`figma-auth.css`):** Implementation of the UVT Authentication Interface exported from Figma design specifications.
